This is a small project where users can generate a personalized ticket for a coding conference. You can upload your avatar, enter your name, email, and GitHub username, and get a cool ticket with your details and a random ticket number.

FEATURES:
Upload an avatar and preview it,
Enter your full name, email, and GitHub username,
Form validation for empty fields and email format,
Generates a personalized ticket with:
Your avatar,
Your name and GitHub username,
Date and city (fetched from IP API),
Random ticket number.
Responsive design,
Smooth animations when ticket appears.

HOW TO USE:
Open index.html in your browser,
Fill out the form with your avatar, name, email, and GitHub username,
Click Generate My Ticket,
See your ticket appear with all your details.

TECHNOLOGIES USED:
HTML
CSS (flexbox, grid, and animations)
JavaScript (form validation, DOM manipulation, API fetch)

NOTES:
Currently frontend-only, no backend.
Can be improved by:
Sending ticket details via email,
Adding more animations,
Improving mobile responsiveness.
